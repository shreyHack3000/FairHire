# FairHire — Team Docs

This folder contains team resources. Start here before writing any code.

## Roadmap
`fairhire_roadmap.jsx` — interactive build roadmap for all 4 team members.
Every task, step, tip, and deliverable for every phase is inside.

**How to open:**
1. Go to https://stackblitz.com/fork/react
2. Replace App.jsx content with fairhire_roadmap.jsx content
3. Run — roadmap opens in browser
